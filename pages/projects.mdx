## 2026

<h3 id="finale-ligure-trail-map">Finale Ligure Trail Map</h3>

<Image
  src="/images/projects/fl-trails.png"
  alt="Finale Ligure Trail Map"
  width={800}
  height={450}
  className="next-image"
/>

A Next.js app for exploring mountain biking trails in Finale Ligure, Italy with real-time open/closed status. Built with Leaflet for interactive maps, Supabase for trail data and status updates, and React Query for data fetching. Features filtering by area, difficulty (Singletrail scale), and trail status.

- [GitHub](https://github.com/ASchwad/fl-trail-map)
- [Live App](https://fl-trail-map.vercel.app/)

<h3 id="spotify-playlist-analyzer">Spotify Playlist Analyzer</h3>

A React app that uses the Spotify API to analyze your playlists. Features audio feature analysis with pie chart visualizations, genre tagging, and playlist insights. Built with React, TypeScript, and Tailwind CSS.

- [GitHub](https://github.com/ASchwad/spotify-react-app)
- [Live App](https://aschwad.github.io/spotify-react-app/)

<h3 id="solarkraft">Solarkraft</h3>

<Image
  src="/images/projects/solarkraft.jpeg"
  alt="Solarkraft"
  width={800}
  height={450}
  className="next-image"
/>

A React app that calculates solar metrics for German cities using the Marktstammdatenregister API from the German Bundesnetzagentur. Uses a Cloudflare Worker as a CORS proxy to handle browser same-origin policy restrictions.

- [GitHub](https://github.com/ASchwad/solarkraft-erkunden)
- [Live App](https://aschwad.github.io/solarkraft/)

<h3 id="route-builder">Route Builder</h3>

<Image
  src="/images/projects/route-builder.gif"
  alt="Route Builder"
  width={800}
  height={450}
  className="next-image"
/>

A React/TypeScript app using Leaflet for planning cross-country running routes. Features drag-and-drop waypoints both in the map and controls panel, distance calculation, and GPX file export. Uses Geoapify for map tiles.

- [GitHub](https://github.com/ASchwad/route-builder)
- [Live App](https://aschwad.github.io/route-builder/)

### 2019

<Image
  src="/images/projects/gtc.gif"
